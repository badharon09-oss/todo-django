# To-Do List Web App (Capstone Project)

**Project:** To-Do List Web App  
**Owner:** <Ronald Badaaki Abaasa>  
**Status:** In progress

## Project Overview
A simple task management web application built with Django. Users can sign up, create tasks, update them, mark them complete, and delete them. This project is a capstone assignment and demonstrates backend design, API endpoints, and basic frontend views.

## Features (planned / implemented)
- User authentication (signup/login/logout)
- Task CRUD (create, read, update, delete)
- Mark tasks as completed
- Optional: categories, search, filters
